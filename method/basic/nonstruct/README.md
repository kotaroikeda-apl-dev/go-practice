# 構造体以外の型のメソッド

このディレクトリでは、構造体以外の型にメソッドを定義する方法を学びます。

## 重要なポイント

- Go では、構造体だけでなく、**パッケージ内で定義した型**にメソッドを定義できます
- 組み込み型（`int`、`float64`など）や他パッケージの型には直接メソッドを定義できません
- エイリアス型（`type MyType float64`など）を作成することで、メソッドを定義できます

## この例で学ぶこと

1. **エイリアス型**: `type Temperature float64`

   - `float64`のエイリアス型を定義
   - この型に対してメソッドを定義できる

2. **非構造体型のメソッド**: `func (t Temperature) Celsius() float64`

   - 構造体と同じように、エイリアス型にもメソッドを定義できる
   - レシーバーは値レシーバーでもポインタレシーバーでも可能

3. **型変換**: `float64(t)`

   - エイリアス型から元の型への変換が必要な場合がある

4. **String()メソッド**: 構造体と同じように、非構造体型にも`String()`メソッドを定義できる

## 実行方法

```bash
go run main.go
```
